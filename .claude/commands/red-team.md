# Phase 2: 레드팀 비판적 분석

주제: $ARGUMENTS

Phase 2의 비판적 사고 및 사각지대 발굴만 독립 실행합니다.

## 실행 절차

### Step 1: 자료 수집
- WebSearch로 주제에 대한 다각도 검색 수행
- 특히 비판적 시각, 반대 의견, 실패 사례를 집중 검색
- 학술 비판, 업계 반론, 규제 이슈 등 포함

### Step 2: Task 2.1 - 레드팀식 약점 공격
가장 엄격한 동료 평가자 관점에서 무자비하게 비판:
| 치명적 약점 | 약점 공격 | 심각도 (1-10) | 보완 필요 데이터 |

### Step 3: Task 2.2 - 숨겨진 전제 조건 역추적
명시되지 않은 숨은 가정 파악:
- **숨은 가정** + **중요도/오류 가능성** + **가정 파괴 시나리오**

### Step 4: Task 2.3 - 공백(White Space) 탐색
누락된 심층 질문 5-7가지 생성:
- 패러다임 변환 잠재력이 높은 질문 우선
- 미충족 수요와의 연관성 분석

## 출력
`research-output/phase2/` 디렉토리에 결과 저장
