# Phase 1: 심층 분석 실행

주제: $ARGUMENTS

Phase 1의 심층 분석 및 논리 검증만 독립 실행합니다.

## 실행 절차

### Step 1: 자료 수집
- WebSearch로 주제에 대한 다각도 검색 수행
- 최소 15개 이상의 소스 확보
- 학술/업계/언론 등 다양한 출처 포함

### Step 2: Task 1.1 - 핵심 인사이트 및 통념 타파
해당 분야 전문가 관점에서 핵심 통찰 3-5가지 도출:
- **핵심 통찰** + **타파하는 기존 통념** + **현재 시점의 중요성**

### Step 3: Task 1.2 - 논리적 엄밀성 및 근거 강도 평가
주요 주장별 근거 강도 4단계 분류:
| 주요 주장 | 근거 수준 | 🚩 Red Flag | 평가 이유 | 출처 |

### Step 4: Task 1.3 - 데이터 교차 검증
소스 간 모순/충돌 식별:
- **충돌 지점** + **양쪽 입장** + **배경 요인 추론**

## 출력
`research-output/phase1/` 디렉토리에 결과 저장
